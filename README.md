# IJA 2025/26 Project

Course project inspired by Advance Wars.

## Assignment Snapshot

- Build a turn-based strategy game on an `M x N` grid with terrain, units, economy, combat, capturing, logging, and replay.
- Keep the game engine separate from the UI using MVC.
- Support at least one simple AI opponent and a `Bot vs. Bot` mode.
- Use git continuously and keep commit history meaningful for all team members.

## Repository Layout

- `assignment-summary.md` contains a condensed project analysis based on the assignment.
- `docs/` is ignored locally, so raw assignment assets can stay on your machine without being committed.

## Immediate Next Steps

1. Choose whether the UI will be Swing or JavaFX.
2. Define the domain model for map tiles, units, buildings, players, and turns.
3. Implement pathfinding and turn execution in the engine before building UI interactions.
